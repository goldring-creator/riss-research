# 📚 RISS 논문 수집기

RISS(학술연구정보서비스)에서 키워드로 국내학술지 논문을 자동 수집하고 PDF를 다운로드하는 데스크탑 앱입니다.

> ⚠️ **학위논문은 수집되지 않습니다.** 국내 학술지 논문만 대상으로 합니다.  
> ⚠️ **macOS 전용**입니다. (Windows 미지원)

---

## 실행 화면

단일 화면에서 키워드 추가, 필터 설정, 저장 위치 지정 후 수집 시작.

```
📚 RISS 논문 수집기          ⚠️ 학위논문은 포함되지 않습니다

🔐 도서관 로그인 설정
  도서관 ID / PW 입력 후 [ID 저장]

🔍 검색 키워드
  [교사 정체성 ×] [교사 정체성 연구 ×] [  입력...  ]

⚙️ 필터 및 옵션
  페이지: 3 | 연도: 2015 ~ 2024 | KCI 등재만 ☑

📁 저장 위치
  /Users/.../바탕화면/논문모음  [폴더 선택]

[▶ 수집 시작]
```

---

## 설치 방법

### 1. 사전 요구사항

**Node.js 설치 여부 확인** — 터미널에서:
```bash
node -v
```
버전 번호가 뜨면 OK. 없으면 [nodejs.org](https://nodejs.org) 에서 LTS 버전 설치.

**Playwright 브라우저 설치** (최초 1회):
```bash
npx playwright install chromium
```

---

### 2. 다운로드

이 페이지 상단 초록색 **Code** 버튼 → **Download ZIP** → 압축 해제

또는 터미널:
```bash
git clone https://github.com/goldring-creator/riss-research.git
```

---

### 3. 패키지 설치

압축 해제한 폴더 안에서 터미널 실행 후:

```bash
npm install
```

---

### 4. 실행

```bash
node launcher/server.js
```

터미널에 `http://127.0.0.1:47281` 이 뜨면 브라우저에서 해당 주소 열기.

> 💡 매번 터미널을 열기 번거롭다면, 관리자에게 `.app` 런처 파일 요청.

## 사용 방법

### ① 도서관 ID/PW 저장

- 소속 도서관의 이용자 ID와 PW를 입력하고 **[ID 저장]** 클릭
- 비밀번호는 macOS Keychain에 암호화 저장됩니다
- 처음 저장 시 시스템 팝업 → **"항상 허용"** 클릭

### ② 키워드 입력

- 검색할 키워드를 입력 후 **Enter** 또는 **쉼표(,)**
- 여러 키워드를 추가하면 각각 검색 후 결과를 합쳐 중복 제거

### ③ 필터 설정 (선택)

| 옵션 | 설명 |
|------|------|
| 수집 페이지 수 | 1페이지 = 약 10건 |
| 연도 범위 | 빈칸이면 전체 |
| KCI 등재만 | 체크 시 KCI 등재 논문만 |
| 정렬 | 정확도순 / 최신순 / 인기도순 |
| 상위 N건 | 피인용수 기준 상위 N건만 |
| 제외 키워드 | 제목/초록에 해당 단어 포함 시 제외 |

### ④ 저장 위치

- **[폴더 선택]** 버튼으로 PDF가 저장될 폴더 지정
- 기본값: 바탕화면/RISS_출력

### ⑤ 수집 시작

- **[▶ 수집 시작]** 클릭
- 하단 로그에서 실시간 진행 상황 확인
- 도중 중단하려면 **[⏹ 중단]** 클릭

## Claude API 분류 기능 (선택)

수집된 논문을 주제별로 자동 분류하는 기능입니다.

1. [console.anthropic.com](https://console.anthropic.com) 에서 API 키 발급
2. 앱의 **"Claude API 키"** 입력란에 입력 후 **[키 저장]**
3. 필터 패널의 **"분류 건너뜀"** 체크 해제

## 자주 묻는 질문

**Q. 도서관 로그인이 실패합니다.**  
A. 소속 기관 도서관 ID/PW인지 확인하세요. RISS 직접 로그인 계정이 아닌, 기관 도서관 계정을 사용해야 합니다.

**Q. PDF 다운로드가 안 되는 논문이 있습니다.**  
A. 일부 논문은 제공사 정책상 원문 접근이 불가합니다. `no_link`로 표시됩니다.

**Q. 학위논문을 수집하고 싶습니다.**  
A. 이 앱은 국내 학술지 논문만 지원합니다. 학위논문 수집은 지원하지 않습니다.

**Q. Keychain 팝업이 매번 뜹니다.**  
A. 처음 뜰 때 "허용" 대신 **"항상 허용"** 을 클릭하면 이후 팝업이 뜨지 않습니다.

## 주의사항

- 이 앱은 소속 기관 도서관 계정을 통해 RISS에 접근합니다.
- 수집된 논문은 개인 연구 목적으로만 사용하세요.
- 단시간에 대량 수집 시 서버 부하로 수집이 느려질 수 있습니다.
